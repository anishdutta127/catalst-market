# AI Build Rules

- Make the prompt specific enough for Codex, Cursor, Lovable, or Bolt.
- Include the target user, exact problem, waitlist offer, and acceptance criteria.
- Avoid full dashboards in v1.
- Avoid auth unless the waitlist truly requires it.
- Add waitlist capture first.
- Keep the page fast, readable, and mobile-first.
- Use one CTA.
- Include a short list of what not to build yet.
