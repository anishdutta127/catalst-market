# Waitlist Validation

- Capture email first.
- Track waitlist form submissions.
- Track primary CTA clicks.
- Ask one qualifying question that proves the user matches the target group.
- Message real users within 48 hours.
- Ask about current workarounds before asking for feature ideas.
- Count replies, calls booked, and waitlist joins.
- Do not build the full product until the offer gets real interest.
